​レッツエンジョイ東京
サイト​リデザイン



2020.
Personal Project
Language : Japanese

​​既存のサイトイメージ
https://www.enjoytokyo.jp/

​情報中心のサイトであるため、説明文が多いのは
当然のことだが、情報に興味を持っている立場から
見たとき、テキスト中心のサイトは届かなかったり、
速く伝わらなかったりする場合が多い。 画像をさらに強調して、
サイトに接続したユーザーが情報により
アクセスしやすくなるように再構成してみた。

​​全体的にイメージのサイズを大きく変更し、テキスト全てを読まなくても、
イベントの雰囲気をキャッチできるようなイメージトリミングを行いました。
各メニューのイメージの形も多様化させ、ダイナミックな雰囲気を演出しました。
東京を象徴する物や、メニューのテーマを表すイラストをページに追加し、
サイトに入った方を飽きさせないように工夫しました。


Jo Loves Website Redesign

​ジョーラブズ サイト​リデザイン



2020.
Personal Project
Responsive Web
Language : Japanese

​​既存のサイトイメージ
https://www.joloves.com/

​現在のサイトでは、反応型ウェブでシンプルさを強調し、
写真画像だけで商品をリストアップしていますが、
ブランドのさまざまなテーマの製品を表示するためには、
よりデザイン的な要素が必要であると感じ、香りの感覚を線と面で表現し、
より動的な感じのイメージに再構成してみた。

​​シンプルなデザインだった既存のホームページを、もっとカジュアルな雰囲気で再構成しました。
ランディングページでは、様々ななコンセプトの商品を一目で見ることができるように並べ、
ブランドのストーリーと、購入にリンクされたボタンを一緒にレイアウトしています。
曲線のラインの中に、創業者の一言をカリグラフィーのテキストでインサートし、
カジュアルなイメージを活かしながら、親しみを感じるデザインに演出しました。

ハンターブーツ サイト​リデザイン



2020.
Personal Project
Language : English

​​既存のサイトイメージ
https://www.hunterboots.com/us/en_us/

シンプルな雰囲気と大きな画像で構成されているが、
ブランド自体のイメージを連想させるには不十分であると
判断した。購入誘導に焦点を当てており、 ブランドストーリー、歴史、
サステナビリティや社会貢献の内容を一目で見るのは難しい点があった。

​​画像を強調するレイアウトは維持しながら、
ブランドのストーリー、コンセプトなどがすぐにユーザーに伝わるように、
写真とレイアウトを再構成してみた。最近はブランドの価値観により、
消費者の購入につながる 消費パターンが増加しているため、そのメニューのレイアウトを変更し、
ランディングページから見ることができるように修正した。
ランディングページでは、ブランドストーリーを確認ボタンと共にに商品購入に繋がるボタンを
配置しました。カラーは既存のロゴのメインカラーに、水を連想させるカラーを加え構成しました。
